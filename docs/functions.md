## Functions

zsh tmux includes a small utility `tmux`.

#### `tmux::post_install`

Synchronizes and install configurations.

#### `tmux::sync`

Synchronizes the configurations that are in `conf/.tmux.conf`

#### `tmux::install`

Install tmux

#### `ftm`

Create new tmux session, or switch to existing one.

#### `ftmk`

Delete tmux session.
